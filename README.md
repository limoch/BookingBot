# Telegram Booking Bot

A **modern and flexible Telegram bot** template for appointment scheduling. Designed for salons, studios, and service providers, this bot allows clients to book services, view portfolios, and receive reminders, all in a clean, easy-to-use interface.

Perfect for nail salons, hair studios, spas, or any service-based business.

---

## Features

### Client-side
- Showcase your portfolio with images
- Book appointments with available time slots
- Automatic reminders 1 hour before the appointment
- View all your bookings in one place
- Clean interface: all interactions happen in a single message

### Admin-side
- View all bookings
- Delete appointments easily
- Admin panel with streamlined menu

### UX Enhancements
- Navigation buttons for going back
- Prevent duplicate bookings for the same service
- Fully customizable for different services

## Installation

Follow these steps to set up the bot:

1. **Clone the repository**

```bash
git clone https://github.com/limoch/BookingBot.git
```
2. Create a virtual environment
```bash
python -m venv venv
# Activate on Windows
venv\Scripts\activate
# Activate on Linux/macOS
source venv/bin/activate
```
Install dependencies
```bash
pip install -r requirements.txt
